# Interesting Hardware Security Talks & Videos

This page contains a curated list of noteworthy hardware security presentations, talks, and video content from conferences, YouTube channels, and other sources.

## Conference Talks

### DEF CON Talks

#### "The Year in Hardware Hacking" - Joe FitzPatrick
- **Description**: An overview of significant hardware vulnerabilities and techniques discovered over the past year.
- **Link**: [YouTube](https://www.youtube.com/watch?v=Y1X61xH_7A0)

#### "Glitching RISC-V chips" - Dmitry Nedospasov & Josh Datko
- **Description**: Demonstration of fault injection attacks against RISC-V architecture chips.
- **Link**: [YouTube](https://www.youtube.com/watch?v=0HxT8j7cW6M)

#### "Hardware Implants in the Supply Chain" - Sarah Edwards
- **Description**: Explores the threats posed by hardware implants inserted during manufacturing and distribution.
- **Link**: [YouTube](https://www.youtube.com/watch?v=C-YRIvXpLb0)

### Black Hat Talks

#### "Applied Physical Attacks on Embedded Systems" - Joe Grand
- **Description**: Practical demonstration of physical attacks against embedded systems.
- **Link**: [YouTube](https://www.youtube.com/watch?v=nkCsBKcMcpw)

#### "Breaking Hardware-Enforced Security with Hypervisors" - Alex Matrosov
- **Description**: Demonstrates bypassing hardware security features using virtualization.
- **Link**: [YouTube](https://www.youtube.com/watch?v=nvV_4U9cFn0)

### Hardwear.io Talks

#### "Practical Hardware Attacks on Deep Learning" - Maria Markstedter
- **Description**: Explores how hardware vulnerabilities can be used to compromise deep learning systems.
- **Link**: [YouTube](https://www.youtube.com/watch?v=89QPY2armyM)
