## Setting Up Your First Hardware Security Lab

### Beginner Lab Setup (Under $200)

1. **Basic Tools**
   - Digital multimeter ($30)
   - Basic soldering iron kit ($30)
   - USB-to-serial adapter ($10)
   - Logic analyzer ($10-30)
   - Breadboard and jumper wires ($15)

2. **Target Hardware**
   - Arduino/ESP32 development boards ($10-20)
   - Raspberry Pi Zero W ($15)
   - Old routers/IoT devices from thrift stores ($5-20 each)

3. **Software Tools (Free)**
   - Ghidra (reverse engineering)
   - Sigrok/PulseView (logic analysis)
   - Arduino IDE (firmware development)
   - Various serial terminal programs

### Resources for Lab Setup

- [Build a Hardware Hacking Lab on a Budget](https://hackaday.com/2019/05/07/build-a-hardware-hacking-lab-on-a-budget/) - Hackaday guide
